Built for AP teams

Automate the tedious parts of accounts payable classification so you can focus on exceptions.

LLM Classification

Routes transactions through Claude, GPT, and other models via OpenRouter for accurate multi-level taxonomy classification.

Supplier Research

Automatically researches unknown suppliers on the web to understand what they do, improving classification accuracy.

Custom Taxonomy

Define your own 3-level classification hierarchy (L1 → L2 → L3) to match your chart of accounts or reporting needs.

Override Rules

Set up rules to auto-classify recurring suppliers or transaction patterns without using LLM credits.

QC & Review

AI-powered quality checks flag low-confidence results. Review, correct, and approve classifications before export.

Portable Projects

Each project is a single .c91 file powered by DuckDB. Share projects with colleagues by sending one file.

How it works

1

Import your data

Upload a CSV of AP transactions. The app auto-maps your columns to canonical fields like supplier name, amount, and GL description.

2

Configure taxonomy & context

Import or define your classification taxonomy and set up company context so the AI understands your business.

3

Run AI classification

The app researches unknown suppliers, then classifies each transaction using LLMs. Override rules handle known patterns instantly.

4

Review & export

Review flagged transactions, correct any misclassifications, then export the final results to CSV.
